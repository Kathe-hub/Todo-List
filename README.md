# Todo-List
## Lista de pendientes

Creación de sitio web orientado a convertir monedas desde pesos chilenos a dolares y euros. Incluye un gráfico que muestra las variaciones de ambas divisas los últimos 10 días.

Tecnologías utilizadas:
- _**HTML5**_
- _**CSS Avanzado**_ 
- _**Canvas**_
- _**JavaScript**_
